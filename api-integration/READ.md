# API Data Integration

## Purpose
This script retrieves product, order, and inventory data from shopify via REST API to feed the Power BI analytics dashboard.

---

## Functionality
- Authenticates against API
- Retrieves product data
- Retrieves order transaction data
- Performs pagination handling
- Converts data into structured format
- Prepares data for BI reporting

---

## Technologies
- Python
- Requests Library
- JSON Processing

---

## Architecture Flow
API →Advanced Query Power BI → Power BI → Dashboard

---

## Data Privacy
All sensitive credentials removed.
